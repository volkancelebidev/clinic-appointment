# Clinic Appointment Management System

A lightweight clinic appointment management system backed by SQLite.

## Features
- Doctor, Patient and Appointment management
- BMI calculation and risk classification
- Specialty-based statistics with completion rates
- High-risk patient detection (BMI >= 30)
- CSV and JSON export

## Tech Stack
- Python 3.12
- SQLite3 (built-in sqlite3 module)

## How to Run
python clinic_appointment.py
